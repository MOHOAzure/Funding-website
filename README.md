# Funding-website
A django website for funding projects

![](https://i.imgur.com/SCzH9MR.png)

## Env Requriements
- Python 2.7
- Django 1.11

## Run server steps
  - <opt.> create a user
  - python manage.py makemigrations funding
  - python manage.py migrate --run-syncdb
  - python manage.py runserver

## Webpages
- Admin page: ^admin 
- Login page: ^login
- Funding page: ^funding
